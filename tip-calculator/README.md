# 💰 Tip Time

A simple Android app built with **Jetpack Compose** to calculate tips quickly and easily.

---

## 🚀 Features

- Enter bill amount
- Set tip percentage
- Toggle rounding
- View tip in local currency format

---

## 🛠 Tech

- Kotlin
- Jetpack Compose (Material 3)
- JUnit (local tests)
- Compose UI Test (instrumentation tests)
